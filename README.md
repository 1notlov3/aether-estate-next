# Aether Estate — Next.js 16

Современный лендинг по продаже недвижимости на **Next.js (App Router)**:

- 3D hero parallax
- кастомный курсор
- tilt-эффекты карточек
- magnetic-кнопки
- premium dark-theme
- локальные сгенерированные SVG-ассеты в `public/assets`

## Запуск локально

```bash
npm install
npm run dev
```

Открыть: `http://localhost:3000`

## Прод

```bash
npm run build
npm run start
```
